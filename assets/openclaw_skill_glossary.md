# 📖 OpenClaw技能大白话词典（安全技能版）
## 一、通用术语映射表（技术术语→自然语言）
| 技术术语 | 大白话解释 | 适用场景 |
|----------|------------|----------|
| `skill installer` / `clawhub install` | 技能安装器 | 你要装新功能/新工具的时候用 |
| `self improving agent` | 自我进化智能助理 | 想让AI自己学习、自己优化能力的时候用 |
| `tools.md` / `core toolset` | 核心工具包 | 包含所有常用基础功能的工具集合 |
| `cron job` / `scheduled task` | 定时任务 | 要设置每天/每周自动干的活（比如定时发报告、定时巡检） |
| `multi-brain orchestration` | 多脑并行工具 | 要同时干好几个复杂任务、提升效率的时候用 |
| `debug-pro` | 自动排障工具 | 代码/系统出问题了自动找bug修bug的时候用 |
| `pdf-processor` | PDF处理工具 | 要读PDF、提取PDF内容、扫描件转文字的时候用 |
| `ontology` | 知识图谱工具 | 要整理知识库、做实体关联分析的时候用 |
| `gcalcli-calendar` | 日历提醒工具 | 要设置日程提醒、同步日历的时候用 |
| `pa-admin-exec` | 行政助理工具 | 要自动处理邮件、整理会议纪要、做日程管理的时候用 |
